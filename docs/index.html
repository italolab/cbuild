O que é?


O cbuild é um software de linha de comando que pode ser utilizado para automatizar o processo de build de projetos C/C++, permitindo configurar tarefas e executá-las para ter como resultado final: um executável linkado, uma shared library, uma static library, execução de testes unitários, etc.
