Procedimentos

Você também pode criar seus próprios procedimentos e chamá-los nos blocos das definições de tarefa ou qualquer bloco de código no script. Para definir um procedimento utilize "proc" e "endproc" e para chamar o procedimento utilize o comando "call". Veja o exemplo abaixo: 

proc imprimeMensagemDeCompile
    echo A compilação está sendo executada...
endproc

task compile before
    call imprimeMensagemDeCompile
endtask

task compileall before
    call imprimeMensagemDeCompile
endtask

No exemplo acima, foi definido um procedimento com nome: "imprimeMensagemDeCompile" e ele é chamado nas definições "task compile before" e "task compileall before".
